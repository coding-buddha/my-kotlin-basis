# springboot-websocket-basis

## 준비
* http://localhost:8080 을 실행한다.
* `http://localhost:8080/index.html` 을 입력하여 대화형 웹소켓 수행을 한다.
* `http://localhost:8080/index.html` 을 입력하여 스케줄링형 웹소켓 수행을 한다.
